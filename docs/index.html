Welcome to TinyLLM’s documentation!

TinyLLM is a lightweight, semantic-augmented language model trained from scratch on a Quadro K620. It features:

  • Subword/BPE via GPT-2 tokenization and semantic-augmented training

  • Learnable per-layer gates for semantic influence

  • Adaptive regulation with arithmetic compensation

  • Sharded data loading for large datasets

  • Context blending for local coherence
